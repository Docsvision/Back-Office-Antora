Карточка _Бизнес-календарь_ может быть создана различными способами. Однако, все способы создания предполагают выбор вида создаваемого объекта. В базовый комплект поставки системы _{dv} 5_ (при отсутствии модуля _{dm}_) входит базовый тип карточки _Бизнес-календарь_.
