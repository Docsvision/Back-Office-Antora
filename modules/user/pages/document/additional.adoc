= Дополнительные возможности карточки "Документ"

При внедрении системы {dv} в базовую разметку карточки _Документ_ могут быть добавлены дополнительные элементы управления, значительно расширяющие возможности карточки. Полный список доступных элементов содержится в _Конструкторе разметок_.

.Некоторые элементы управления, используемые для расширения функциональности карточки Документ:
* xref:additional/additional-polls.adoc[Настройка голосования].
* xref:additional/additional-performance-tree.adoc[Настройка управления связанными карточками].
* xref:additional/additional-links.adoc[Настройка отображения связанных карточек].
