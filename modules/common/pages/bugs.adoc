= Исправленные ошибки

include::webclient:admin:partial$table.adoc[]
|ERR-1233 (SUP-5762)
|Возникала ошибка `An item with the same key has already been added` при просмотре и последующем закрытии элемента управления `_Назначенные исполнители_`.

|ERR-1114 (SUP-5701)
|Исправлена ошибка, при которой отсутствовали кнопки и задний фон в письме почтового клиента.

|ERR-842 (SUP-5502)
|При сериализации поля `Decimal` в XML происходила ошибка `System.NotSupportedException: Don't know how to safe convert value '9000,00000000000000' of type Decimal to string` из-за которой выгружались не все секции карточки при использовании REST Object Manager.

|ERR-785 (SUP-5457)
|Исправлена ошибка, когда запрещенная по ролевой операция _Удалить_ не позволяла удалить карточку версии файла.

|ERR-356 (SUP-5434)
|При использовании маршрутизации *_Письмо с описанием задания_* применялся тот же вид письма, что и при маршрутизации *_Задача почтового клиента_*.
