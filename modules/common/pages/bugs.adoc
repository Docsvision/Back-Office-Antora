= Исправленные ошибки

include::webclient:admin:partial$table.adoc[]
|ERR-1914 (SUP-6260)
|Исправлена ошибка, при которой метку в элементе управления `_Да / Нет_` было невозможно скрыть.

|ERR-1798 (SUP-6164)
|При просмотре определённых свойств в структуре разметок, {wincl} или РМА переставали отвечать.

|ERR-1685 (SUP-6082)
|Исправлена ошибка, при которой в журнал {wincl}а записывалось: `Error occurred when calling Cryptography service with address`.

|ERR-1668 (SUP-5970)
|Было возможно завершить задание на подписание с использованием УКЭП без ошибки, если просрочена лицензия CSP.

|ERR-1514 (SUP-5984)
|В файловом хранилище оставались удалённые из карточки файлы, если файлы были удалены из карточки до её сохранения.

|ERR-1280 (SUP-5798)
|Исправлена ошибка, при которой если у группы заданий нет собственного списка заданий (TaskList), нарушалась логика вычисления подчиненных заданий и групп заданий.

|ERR-1279 (SUP-5819)
|При работе с согласованием через почтовый клиент, в "Ходе согласования" не отображались результаты делегирования.

|ERR-1058 (SUP-5680)
|При разворачивании узла конструктора справочников область со списком узлов прокручивалась до начала.

|ERR-894 (SUP-5558)
|Возникала ошибка `Unable to initialize security context extension` при использовании sql-скрипта в бизнес-процессе.

|ERR-1233 (SUP-5762)
|Возникала ошибка `An item with the same key has already been added` при просмотре и последующем закрытии элемента управления `_Назначенные исполнители_`.

|ERR-1114 (SUP-5701)
|Исправлена ошибка, при которой отсутствовали кнопки и задний фон в письме почтового клиента.

|ERR-842 (SUP-5502)
|При сериализации поля `Decimal` в XML происходила ошибка `System.NotSupportedException: Don't know how to safe convert value '9000,00000000000000' of type Decimal to string` из-за которой выгружались не все секции карточки при использовании REST Object Manager.

|ERR-785 (SUP-5457)
|Исправлена ошибка, когда запрещённая по ролевой операция _Удалить_ не позволяла удалить карточку версии файла.

|ERR-471 ()
|При согласовании из почты не передавались комментарии в {wc}.

|ERR-417 (SUP-5104)
|Возникала ошибка при смене состояния карточки документа с помощью скрипта: `Unable to set properties to signed state`.

|ERR-356 (SUP-5434)
|При использовании маршрутизации *_Письмо с описанием задания_* применялся тот же вид письма, что и при маршрутизации *_Задача почтового клиента_*.
