* xref:card-kinds/Common_Hide_subtype.adoc[Доступность вида для выбора];
* xref:card-kinds/Common_Inheritance.adoc[Настройки наследования];
* xref:card-kinds/CreateBisnesProcess.adoc[Процессы];
* xref:card-kinds/SetCardCreationMode.adoc[Способ создания карточек].
