= Performing

== Назначение

Группа элементов отображает панель *Ход исполнения*.

Группа включает несколько сгруппированных элементов:

* *Percent completed*;
* группа *Actual performing date*;
* *Completed user*;
* *Текущие исполнители*;
* группа *Comments*.
