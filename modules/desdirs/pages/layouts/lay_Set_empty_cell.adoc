= Создание пустой ячейки

Элемент xref:layouts/lay_Elements_EmptySpace.adoc[Пустая ячейка] используется при настройке отображения элементов управления в разметке карточки. Согласно правилам размещения элементов в разметке, если ячейка элемента единственная в своей "строке", она по умолчанию занимает всю ширину карточки. При помощи пустых ячеек занимаемое элементами пространство может быть ограничено.

.Чтобы создать пустую ячейку, выполните любое из действий:
* в области _Разметки_ или области _Структура разметки_ элемент вызовите контекстное меню, затем выберите команду *Создать пустую ячейку*;
* перетащите из области _Панель инструментов_ в область _Разметки_ элемент xref:layouts/lay_Elements_EmptySpace.adoc[Пустая ячейка].

Элемент будет добавлен в разметку карточки.
