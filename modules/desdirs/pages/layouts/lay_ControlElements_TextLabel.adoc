= Настройка текстовой метки

Для каждого элемента, имеющего большую форму свойств, можно настраивать расположение текстовой метки.

Настройка метки выполняется настройкой следующих xref:layouts/lay_Elements_general.adoc[Общих свойств элементов управления]:

* xref:layouts/lay_Elements_general.adoc#align[Автовыравнивание];
* xref:layouts/lay_Elements_general.adoc#visibility[Видимость];
* xref:layouts/lay_Elements_general.adoc#placement[Местоположение].
