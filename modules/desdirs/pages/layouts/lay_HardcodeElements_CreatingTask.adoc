= Creating Task

== Назначение

Группа элементов отображает панель *Создание задания*.

.Группа включает элементы:
* Группа xref:layouts/lay_HardcodeElements_Performers.adoc[*Performers*];
* Группа xref:layouts/lay_HardcodeElements_Inspection.adoc[*Inspection*];
* Группа *Сроки*;
* *Редактор даты напоминания*;
* *Редактор значения напоминания*;
* *Напомнить за несколько часов*;
* *Дата напоминания*;
* *Часы до завершения задания*;
* *Content*;
* *Name*;
* *Author*.
