= Main

== Назначение

Группа элементов отображает панель *Основная*.

.Группа включает элементы:
* *Performers*;
* *Responsible performer;*;
* *layoutControlItem1*;
* *Контроль до завершения (ч.)*;
* *Controller:*;
* *layoutControlItemOnControl*;
* *Name:*;
* *Content:*;
* *StartDate:*;
* *ControlDate:*;
* *Срок исполнения:*;
* *Duration (h.)*;
* *Author:*;
* *Laboriousness (h.):*;
* xref:layouts/lay_HardcodeElements_Priority.adoc[*Priority:*].
