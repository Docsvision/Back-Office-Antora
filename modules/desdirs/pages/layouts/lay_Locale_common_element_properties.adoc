= Локализация общих свойств "дополнительных" элементов управления

Для всех xref:layouts/lay_Control_elements.adoc[дополнительных элементов управления], включая также вкладки и области группировок, имеется возможность настройки локализации следующих _общих_ свойств:

* xref:layouts/lay_Elements_general.adoc#label[*Текст метки*];
* xref:layouts/lay_Elements_general.adoc#hint[*Подсказка*];
* xref:layouts/lay_Elements_general.adoc#align[*Автовыравнивание метки*];
* xref:layouts/lay_Elements_general.adoc#placement[*Местоположение метки*].

Описание настройки локализации для данных свойств содержится в секции *Настройка локализации* раздела xref:layouts/lay_Elements_general.adoc[Общие свойства элементов управления].
