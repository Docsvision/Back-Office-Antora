= Изображение

== Назначение

Элемент предназначен для размещения в карточке фиксированного изображения, которое может использоваться в декоративных или информационных целях.

== Данные

Источник данных::

== Внешний вид

Изображение::
  Атрибут задает изображение, отображаемое элементом.
